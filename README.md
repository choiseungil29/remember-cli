# remember-cli

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

## 개선 가능한 사항
 - Navbar에 상태에 따라 다르게 나타나도록 구현
 - User Create View와 User Update View 분리
 
## 그 외
 - 최초 관리자 계정은 admin/admin으로 설정되어져 있습니다.
